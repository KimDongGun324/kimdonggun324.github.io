## 1. Dictionary (딕셔너리)

### 개념
- `{key: value}` 형태의 자료구조
- **key**: 중복 불가, 어떤 타입이든 가능
- **value**: 중복 허용, 어떤 타입이든 가능

### 주요 메서드

| 메서드 | 설명 |
|--------|------|
| `d.keys()` | 모든 key 반환 |
| `d.values()` | 모든 value 반환 |
| `d[key]` | 해당 key의 value 접근 |
| `d[key] = val` | 새 key-value 추가 또는 기존 value 수정 |
| `"key" not in d` | key 존재 여부 확인 (`True`/`False`) |
```python
a = {"A-B": 3.234, "B-C": 2.54}
a["X"] = "new value"      # 새 항목 추가
a["A-B"] = a["A-B"] + 1  # 기존 값 수정
```

## 2. 조건문 (if / elif / else)

### 구조
```python
if 조건1:
    # 조건1이 True일 때
elif 조건2:
    # 조건2가 True일 때
else:
    # 모두 False일 때
```

### 논리 연산자 (Logical Operators)

| 연산자 | 의미 |
|--------|------|
| `a == b` | 같다 |
| `a != b` | 다르다 |
| `a > b`, `a < b` | 크다, 작다 |
| `a >= b`, `a <= b` | 크거나 같다, 작거나 같다 |
| `and` | 둘 다 True |
| `or` | 하나라도 True |
| `not` | True → False 반전 |
```python
score = 75
if score > 90:
    grade = "A"
elif score > 70 and score <= 90:
    grade = "B"
else:
    grade = "F"
```

> **들여쓰기(Indentation)**: Python은 블록을 `{}` 대신 들여쓰기로 구분.  
> 칸 수는 상관없지만 **일관성**이 필수.

## 3. for 반복문 (for loop)

### 기본 구조
```python
for 변수 in 순회가능객체:
    # 반복할 코드
```

### `range()` 함수

| 형태 | 의미 |
|------|------|
| `range(n)` | 0 ~ n-1 |
| `range(a, b)` | a ~ b-1 |
| `range(a, b, c)` | a ~ b-1, c씩 증가 |
| `range(a, b, -1)` | a ~ b+1, 역순 (단, a > b) |
```python
# 1~10 합계
s = 0
for i in range(1, 11):
    s += i  # s = 55

# 홀수만 (1, 3, 5, 7, 9)
for i in range(1, 11, 2):
    print(i)

# 역순 (10, 8, 6, 4, 2, 0)
for i in range(10, -1, -2):
    print(i)
```

### Dictionary 순회
```python
for key in a:
    print(key, a[key])  # key와 value 동시 출력
```

### 중첩 리스트 언패킹 (Unpacking)
```python
layered_list = [[1, 2], [3, 4]]
for i, j in layered_list:  # 원소가 2개이므로 변수도 2개
    print(i, j)
```

### 문자열 순회
```python
for ch in "HELLO":
    print(ch)  # H, E, L, L, O 순서로 출력
```

## 4. while 반복문 (while loop)

### 기본 구조
```python
while 조건:
    # 조건이 False가 될 때까지 반복
```

### `break` — 강제 탈출
```python
c = 1
while True:      # 무한루프
    print(c)
    if c == 10:
        break    # 조건 충족 시 탈출
    c += 1
```

> `while True:` + `break` 패턴은 `while 조건:` 과 **동일한 결과**를 낼 수 있음.  
> 복잡한 탈출 조건이 있을 때 유용.

### 축약 연산자 (Compound Assignment)

| 표현 | 동일한 의미 |
|------|------------|
| `c += 1` | `c = c + 1` |
| `s += c` | `s = s + c` |

## 5. 라이브러리 import

### import 방법 3가지
```python
import math           # math.pi 처럼 모듈명으로 접근
import math as mt     # mt.pi 처럼 별칭(alias)으로 접근
from math import pi   # pi 직접 사용
```

### math
```python
import math
math.pi  # 3.14159265358979...
```

### numpy (`np`)
- 리스트를 **배열(Array)**로 변환해 **원소별(element-wise) 연산** 가능
```python
import numpy as np
x = np.array([1, 2])
y = np.array([2, 3])
x + y  # [3, 5]
x - y  # [-1, -1]
x * y  # [2, 6]
x / y  # [0.5, 0.667]
```

> 일반 Python 리스트: `[1,2] + [2,3]` → `[1, 2, 2, 3]` (이어붙이기)  
> numpy array: 수학적 **벡터 연산**처럼 동작

### random
```python
import random
random.random()       # 0.0 ~ 1.0 사이 무작위 실수
random.randint(1, 6)  # 1 ~ 6 사이 무작위 정수 (주사위)
```

## 📌 Summary

- **Dictionary**: `{key: value}`, key 중복 불가, value 중복 허용
- **조건문**: `if / elif / else` + 논리 연산자 (`and`, `or`, `not`)
- **for loop**: 순회 가능 객체 반복 / `range()`로 수 범위 지정
- **while loop**: 조건이 `False`가 될 때까지 반복 / `break`로 강제 탈출
- **import**: `math` / `numpy`(배열 연산) / `random`
